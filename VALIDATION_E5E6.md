## 📞 En Cas de Problème

1. **Blanc ne s'affiche pas?**
   - Vérifier cache navigateur (Ctrl+Shift+Del)
   - Vérifier lien CSS dans HTML

2. **Texte pas clair?**
   - Vérifier polices system (Chrome/Firefox)
   - Vérifier colors CSS

3. **Responsive cassé?**
   - Vérifier Device Toolbar (F12)
   - Tester 640px strictly

4. **PDFs ne s'ouvrent pas?**
   - Vérifier chemin des fichiers
   - Vérifier Projet/ folder
   - Tester lien direct

5. **Couleurs différentes?**
   - Voir palette #2563eb, #0f172a, etc.
   - Vérifier CSS override
